## User Manual (Ringkas)

### 1) Registrasi
- Buka halaman `/register`
- Isi nama, email, dan password
- Selesaikan pendaftaran

### 2) Login
- Buka halaman `/login`
- Masukkan email dan password

### 3) Diagnosis
- Pilih minimal 3 gejala
- Isi tingkat keyakinan (certainty)
- Sistem menghitung hasil diagnosis dan rekomendasi

### 4) Riwayat
- Buka menu `History` untuk melihat hasil diagnosis sebelumnya
- Export PDF tersedia di halaman hasil

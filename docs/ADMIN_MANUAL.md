## Admin Manual (Ringkas)

### Login Admin
- Akses `/admin`
- Gunakan kredensial admin yang dibuat saat seed data

### Menu Utama
- Dashboard: ringkasan data
- Kelola Penyakit: tambah/ubah/hapus penyakit
- Kelola Gejala: tambah/ubah/hapus gejala
- Kelola Rule: konfigurasi aturan diagnosis
- Riwayat Diagnosis: audit hasil diagnosis
- Pengaturan Sistem: AI provider, rate limit, email, maintenance

### Catatan
- Admin session akan timeout jika tidak aktif.
- Pastikan password admin kuat sebelum production.
